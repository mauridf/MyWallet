# MyWallet

## Sistema de Gestão Financeira Pessoal

API REST profissional construída em .NET 10 seguindo Clean Architecture,
SOLID, DDD e padrões corporativos de mercado.

## Stack

-   .NET 10
-   PostgreSQL
-   Entity Framework Core
-   JWT Authentication
-   Clean Architecture
-   Docker
-   GitHub Actions (CI/CD)
-   Render Deploy
-   Swagger
-   Background Workers

## Features

-   Autenticação JWT
-   Gestão de usuários
-   Contas (Accounts)
-   Transações (Transactions)
-   Dashboard financeiro
-   Fechamento mensal automático
-   Snapshot financeiro
-   Worker de automação
-   Histórico financeiro
-   Auditoria
-   Pipeline real de produção
-   Deploy automático
-   Migrations automáticas

## Infraestrutura

-   Docker multi-stage
-   CI/CD real com GitHub Actions
-   Deploy automático no Render
-   Banco PostgreSQL gerenciado
-   Migrations automáticas no startup

## Arquitetura

-   Domain
-   Application
-   Infrastructure
-   API

Separação real de responsabilidades: - Domain = regra de negócio -
Application = casos de uso - Infrastructure = persistência - API =
exposição
